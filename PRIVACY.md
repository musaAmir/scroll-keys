# Privacy Policy for Keyboard Scroller

Last updated: May 4, 2026

Keyboard Scroller is a Chrome extension that lets users scroll web pages with configurable keyboard shortcuts.

## Data Collection

Keyboard Scroller does not collect, sell, share, or transmit personal information.

The extension does not collect browsing history, page contents, keystroke logs, analytics, telemetry, or any other user data.

## Settings Storage

Keyboard Scroller stores extension preferences using Chrome's `chrome.storage.sync` API. These preferences may include:

- Scroll up and scroll down keys
- Scroll amount
- Smooth scrolling setting
- Animation duration
- Acceleration modifier and multiplier

These settings are stored by Chrome so they can sync across browsers where the user is signed in and has Chrome sync enabled. The extension does not send these settings to any server controlled by the developer.

## Website Access

Keyboard Scroller runs a content script on web pages so it can detect configured keyboard shortcuts and scroll the current page or scrollable panel. The extension does not read, store, or transmit page content.

## Remote Code

Keyboard Scroller does not load or execute remotely hosted code. All extension code is included in the installed extension package.

## Limited Use

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

## Third Parties

Keyboard Scroller does not use third-party analytics, advertising, tracking, or external services.

## Changes

If this privacy policy changes, the updated version will be published with the extension listing or project documentation.

## Contact

For privacy questions, use the support or contact information provided on the Chrome Web Store listing.
